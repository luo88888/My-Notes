## Markdown 基本语法

Markdown 是一种轻量级标记语言，旨在以纯文本格式编写结构化文档，易于阅读和编写，同时可以轻松转换为 HTML 等格式。一下是 Markdown 的常用语法规则和示例。

---

### 1. 标题 (Headings)

- **语法**：在单词或短语前添加井号 (`#`)，井号数量代表标题级别（1 到 6 级，`#` 为一级标题，`######` 为六级标题）。
- **说明**：标题用于构建文档的层级结构，类似于 HTML 中的 `<h1>` 到 `<h6>`。
- **示例**：
  ```markdown
  # 一级标题
  ## 二级标题
  ### 三级标题
  ```

- **渲染效果**：
  # 一级标题
  ## 二级标题
  ### 三级标题

---

### 2. 块引用 (Blockquotes)

- **语法**：在段落前添加大于号 (`>`)。
- **说明**：用于引用文本内容，常用于表示引用的言论或重要说明。
- **示例**：
  ```markdown
  > 这是一段引用的内容。
  ```

- **渲染效果**：
  > 这是一段引用的内容。

#### 2.1 多个段落的块引用

- **语法**：在每个段落前添加 `>`，段间空行也需添加 `>`。
- **示例**：
  ```markdown
  > 第一段引用内容。
  >
  > 第二段引用内容。
  ```

- **渲染效果**：
  > 第一段引用内容。
  >
  > 第二段引用内容。

#### 2.2 嵌套块引用

- **语法**：在需要嵌套的段落前添加多个 `>`，嵌套层级越多，`>` 数量越多。
- **示例**：
  ```markdown
  > 这是第一层引用。
  >> 这是第二层嵌套引用。
  >>> 这是第三层嵌套引用。
  ```

- **渲染效果**：
  > 这是第一层引用。
  >> 这是第二层嵌套引用。
  >>> 这是第三层嵌套引用。

---

### 3. 文本格式

#### 3.1 强调 (Emphasis)

- **语法**：
  - 斜体：用单个星号 (`*`) 或下划线 (`_`) 包裹文本，如 `*斜体*` 或 `_斜体_`。
  - 粗体：用双个星号 (`**`) 或双下划线 (`__`) 包裹文本，如 `**粗体**` 或 `__粗体__`。
  - 粗斜体：用三个星号 (`***`) 或三个下划线 (`___`) 包裹文本，如 `***粗斜体***`。
- **示例**：
  ```markdown
  *这是斜体*
  **这是粗体**
  ***这是粗斜体***
  ```

- **渲染效果**：
  *这是斜体*
  **这是粗体**
  ***这是粗斜体***

#### 3.2 删除线 (Strikethrough)

- **语法**：在文本两端添加双波浪线 (`~~`)，如 `~~删除线~~`。
- **示例**：
  ```markdown
  ~~这是被删除的文本~~
  ```

- **渲染效果**：
  ~~这是被删除的文本~~

#### 3.3 下划线 (Underline)

- **语法**：Markdown 本身不直接支持下划线，但可以通过 HTML 标签 `<u>` 实现。
- **示例**：
  ```markdown
  <u>这是带下划线的文本</u>
  ```

- **渲染效果**：
  <u>这是带下划线的文本</u>

---

### 4. 段落与换行

#### 4.1 段落 (Paragraphs)

- **语法**：段落之间用一个或多个空行分隔。
- **说明**：Markdown 会将连续的文本行合并为一个段落，除非明确分隔。
- **示例**：
  ```markdown
  这是第一个段落。

  这是第二个段落。
  ```

- **渲染效果**：
  这是第一个段落。

  这是第二个段落。

#### 4.2 换行 (Line Breaks)

- **语法**：
  - **自然换行**：在编辑器中按回车键，Markdown 通常保留换行，但在 HTML 渲染时会转换为一个空格。
  - **强制换行**：在行尾添加两个空格后按回车键，或使用 HTML 标签 `<br>`。
- **示例**：
  ```markdown
  这是第一行。  
  这是第二行（强制换行）。
  ```

- **渲染效果**：
  这是第一行。  
  这是第二行（强制换行）。

---

### 5. 列表 (Lists)

#### 5.1 无序列表 (Unordered Lists)

- **语法**：使用星号 (`*`)、加号 (`+`) 或减号 (`-`) 作为列表项标记，后跟空格。
- **示例**：
  ```markdown
  - 项目 1
  - 项目 2
    - 子项目 2.1
    - 子项目 2.2
  - 项目 3
  ```

- **渲染效果**：
  - 项目 1
  - 项目 2
    - 子项目 2.1
    - 子项目 2.2
  - 项目 3

#### 5.2 有序列表 (Ordered Lists)

- **语法**：使用数字加点号 (`.`) 作为列表项标记，后跟空格。
- **说明**：数字不必按顺序编写，Markdown 会自动渲染为连续编号。
- **示例**：
  ```markdown
  1. 第一项
  2. 第二项
     1. 子项 2.1
     2. 子项 2.2
  3. 第三项
  ```

- **渲染效果**：
  1. 第一项
  2. 第二项
     1. 子项 2.1
     2. 子项 2.2
  3. 第三项

#### 5.3 任务列表 (Task Lists)

- **语法**：使用 `- [ ]` 表示未完成任务，`- [x]` 表示已完成任务。
- **示例**：
  ```markdown
  - [x] 完成任务 1
  - [ ] 未完成任务 2
  - [ ] 未完成任务 3
  ```

- **渲染效果**：
  - [x] 完成任务 1
  - [ ] 未完成任务 2
  - [ ] 未完成任务 3

---

### 6. 链接 (Links)

- **语法**：`[链接文本](URL "可选标题")`，其中 `URL` 是链接地址，`"可选标题"` 是鼠标悬停时显示的文本。
- **示例**：
  ```markdown
  [访问 Google](https://www.google.com "Google 首页")
  ```

- **渲染效果**：
  [访问 Google](https://www.google.com "Google 首页")

#### 6.1 相对链接

- **语法**：链接到本地文件或相对路径。
- **示例**：
  ```markdown
  [查看本地文件](./docs/file.md)
  ```

#### 6.2 自动链接

- **语法**：直接输入 URL 或邮箱地址，Markdown 会自动转换为可点击链接。
- **示例**：
  ```markdown
  https://www.example.com
  ```

- **渲染效果**：
  https://www.example.com

---

### 7. 图片 (Images)

- **语法**：`![替代文本](图片路径 "可选标题")`，其中 `替代文本` 是图片加载失败时显示的文字，`可选标题` 是鼠标悬停时显示的文本。
- **说明**：图片语法与链接类似，但前面需加感叹号 (`!`)。
- **示例**：
  ```markdown
  ![图片显示失败](https://via.placeholder.com/150 "示例图片标题")
  ```

- **渲染效果**：
  ![图片显示失败](https://via.placeholder.com/150 "示例图片标题")

---

### 8. 代码 (Code)

#### 8.1 行内代码 (Inline Code)

- **语法**：用反引号 (`` ` ``) 包裹代码内容。
- **示例**：
  ```markdown
  使用 `console.log()` 输出信息。
  ```

- **渲染效果**：
  使用 `console.log()` 输出信息。

#### 8.2 代码块 (Code Blocks)

- **语法**：用三个反引号 (`` ``` ``) 包裹代码块，可在开头指定语言类型以启用语法高亮。
- **示例**：
  ```markdown
  ```javascript
  function hello() {
      console.log("Hello, Markdown!");
  }
  ```
  ```

- **渲染效果**：
  ```javascript
  function hello() {
      console.log("Hello, Markdown!");
  }
  ```

#### 8.3 围栏代码块 (Fenced Code Blocks)

- **语法**：与普通代码块相同，但部分 Markdown 解析器支持额外属性（如行号）。
- **示例**：同上。

#### 8.4 显示行号、高亮显示指定行（看平台）
- highlight=1-3
- highlight=[1-3]
- highlight=[1, 2, 5]

- **示例**
```markdown
```c++ {.line-numbers highlight=[1, 4, 5]}
#include <iostream>

using namespace std;

int main() {
    cout << "Hello world" << endl;
    return 0;
}
```
```
- **效果**
```c++ {.line-numbers highlight=[1, 4, 5]}
#include <iostream>

using namespace std;

int main() {
    cout << "Hello world" << endl;
    return 0;
}
```

### 9. 表格 (Tables)

- **语法**：使用竖线 (`|`) 分隔单元格，使用减号 (`-`) 分隔表头和内容行。
- **对齐方式**：
  - `:-`：左对齐
  - `-:`：右对齐
  - `:-:`：居中对齐
- **示例 1：极简模式**：
  ```markdown
  | 水果名称 | 价格 | 数量 |
  |----------|------|------|
  | 香蕉     | $1   | 5    |
  | 苹果     | $1   | 6    |
  | 草莓     | $1   | 7    |
  ```

- **渲染效果**：
  | 水果名称 | 价格 | 数量 |
  |----------|------|------|
  | 香蕉     | $1   | 5    |
  | 苹果     | $1   | 6    |
  | 草莓     | $1   | 7    |

- **示例 2：课程表**：
  ```markdown
  |          | 周一 | 周二 | 周三 | 周四 | 周五 |
  |---------:|------|------|------|------|------|
  | **1**    | 数学 | 英语 | 数学 | 语文 | 品德 |
  | **2**    | 美术 | 数学 | 品德 | 数学 | 语文 |
  | **3**    | 品德 | 品德 | 语文 | 体育 | 数学 |
  | **4**    | 语文 | 音乐 | 体育 | 品德 | 美术 |
  | **5**    | 数学 | 英语 | 语文 | 数学 | 语文 |
  | **6**    | 品德 | 数学 | 美术 | 劳动 | 活动 |
  ```

- **渲染效果**：
  |          | 周一 | 周二 | 周三 | 周四 | 周五 |
  |---------:|------|------|------|------|------|
  | **1**    | 数学 | 英语 | 数学 | 语文 | 品德 |
  | **2**    | 美术 | 数学 | 品德 | 数学 | 语文 |
  | **3**    | 品德 | 品德 | 语文 | 体育 | 数学 |
  | **4**    | 语文 | 音乐 | 体育 | 品德 | 美术 |
  | **5**    | 数学 | 英语 | 语文 | 数学 | 语文 |
  | **6**    | 品德 | 数学 | 美术 | 劳动 | 活动 |

### 10. 分隔线 (Horizontal Rule)

- **语法**：使用三个或以上的星号 (`*`)、减号 (`-`) 或下划线 (`_`)，行内不能有其他内容，可在符号间加空格。
- **示例**：
  ```markdown
  ***
  ---
  ___
  ```

- **渲染效果**：
  ***
  ---
  ___

---

### 11. 脚注 (Footnotes)

- **语法**：使用 `[^标识]` 标记脚注，在文档末尾或合适位置定义脚注内容 `[^标识]: 脚注内容`。
- **示例**：
  ```markdown
  这是一个脚注示例[^1]。

  [^1]: 这是脚注的详细说明。
  ```

- **渲染效果**：
  这是一个脚注示例[^1]。

  [^1]: 这是脚注的详细说明。

---

### 12. HTML 支持

- **语法**：Markdown 支持直接嵌入 HTML 代码，用于实现 Markdown 不支持的功能。
- **示例**：
  ```markdown
  <div style="color: red;">这是红色文本</div>
  ```

- **渲染效果**：
  <div style="color: red;">这是红色文本</div>

---

### 13. 转义字符 (Escape Characters)

- **语法**：在 Markdown 特殊字符前添加反斜杠 (`\`)，以显示字符本身而非触发语法。
- **示例**：
  ```markdown
  这是星号的转义：\*
  ```

- **渲染效果**：
  这是星号的转义：\*

---

### 总结

1. **兼容性**：不同 Markdown 解析器可能支持的语法有所差异（如任务列表、脚注），建议根据目标平台测试效果。
2. **简洁性**：Markdown 旨在简洁，尽量避免过度嵌套或复杂格式，必要时可结合 HTML。
