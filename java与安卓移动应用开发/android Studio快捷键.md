## Android Studio 常用快捷键

### 1. **代码编辑相关快捷键**
这些快捷键用于代码的编写、格式化、重构等操作。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Ctrl + C`            | 复制                                  | 复制选中的文本或代码。                       |
| `Ctrl + V`            | 粘贴                                  | 粘贴剪贴板中的内容。                         |
| `Ctrl + X`            | 剪切                                  | 剪切选中的文本或代码。                       |
| `Ctrl + Z`            | 撤销                                  | 撤销上一次操作。                             |
| `Ctrl + Shift + Z`    | 重做                                  | 恢复撤销的操作。                             |
| `Ctrl + D`            | 复制当前行                            | 复制当前行并粘贴到下一行。                   |
| `Ctrl + Y`            | 删除当前行                            | 删除光标所在的整行。                         |
| `Alt + Enter`         | 快速修复/导入                        | 显示快速修复建议或自动导入缺少的类。         |
| `Ctrl + Shift + I`    | 快速定义                              | 快速查看方法或变量的定义（不跳转）。          |
| `Ctrl + /`            | 单行注释                              | 为当前行添加或取消单行注释（`//`）。          |
| `Ctrl + Shift + /`    | 块注释                                | 为选中的代码块添加或取消块注释（`/* */`）。   |
| `Ctrl + Shift + Up/Down` | 移动代码行                         | 将当前行或选中的代码块向上或向下移动。       |
| `Ctrl + Alt + L`      | 格式化代码                            | 自动格式化代码，按照代码风格规则缩进。       |
| `Ctrl + Alt + O`      | 优化导入                              | 删除未使用的导入语句，整理导入顺序。         |

### 2. **代码导航相关快捷键**
这些快捷键用于在代码中快速跳转或查找。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Ctrl + Left Click`   | 跳转到定义                            | 点击变量或方法，跳转到其定义位置。           |
| `Ctrl + Alt + Left/Right` | 导航历史                          | 向前或向后导航到之前查看的代码位置。         |
| `Ctrl + G`            | 跳转到指定行                          | 打开“跳转到行”对话框，输入行号后跳转。       |
| `Ctrl + E`            | 最近文件                              | 显示最近打开的文件列表，快速切换。           |
| `Ctrl + Shift + E`    | 最近编辑位置                          | 显示最近编辑的文件位置，快速跳转。           |
| `Ctrl + F`            | 文件内查找                            | 在当前文件中查找文本。                       |
| `Ctrl + Shift + F`    | 全局查找                              | 在整个项目中查找文本。                       |
| `Ctrl + R`            | 文件内替换                            | 在当前文件中替换文本。                       |
| `Ctrl + Shift + R`    | 全局替换                              | 在整个项目中替换文本。                       |
| `Ctrl + N`            | 查找类                                | 快速查找并打开类文件（支持模糊匹配）。       |
| `Ctrl + Shift + N`    | 查找文件                              | 快速查找并打开项目中的文件。                 |
| `Ctrl + Alt + Shift + N` | 查找符号                           | 查找方法、变量等符号（支持模糊匹配）。       |

### 3. **代码补全与提示**
这些快捷键用于代码自动补全和智能提示。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Ctrl + Space`        | 基本代码补全                          | 显示代码补全建议（如变量、方法名）。          |
| `Ctrl + Shift + Space`| 智能补全                              | 显示更智能的补全建议（如匹配类型的参数）。    |
| `Ctrl + P`            | 参数提示                              | 显示方法参数信息（光标在方法括号内时）。     |
| `Ctrl + Q`            | 快速文档                              | 查看方法或类的文档注释（如果有）。           |

### 4. **调试相关快捷键**
这些快捷键用于调试代码，设置断点、运行程序等。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `F5`                  | 进入方法（Step Into）                 | 调试时进入方法内部，逐行执行。               |
| `F6`                  | 跳过方法（Step Over）                 | 调试时执行当前行，但不进入方法内部。         |
| `F7`                  | 跳出方法（Step Out）                  | 调试时跳出当前方法，返回到调用处。           |
| `F8`                  | 继续执行（Resume Program）            | 继续执行程序，直到下一个断点或程序结束。     |
| `Shift + F9`          | 启动调试                              | 以调试模式启动应用。                         |
| `Alt + F9`            | 运行到光标处                          | 运行程序直到光标所在行（需先设置断点）。     |
| `Ctrl + F2`           | 停止调试                              | 停止当前调试会话。                           |
| `Ctrl + Shift + F8`   | 查看断点                              | 打开断点管理窗口，查看所有断点。             |

### 5. **运行与构建相关快捷键**
这些快捷键用于构建项目、运行应用等。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Shift + F10`         | 运行应用                              | 运行当前配置的应用（不调试）。               |
| `Shift + F9`          | 调试应用                              | 以调试模式运行当前配置的应用。               |
| `Ctrl + F9`           | 构建项目                              | 构建整个项目（Make Project）。               |
| `Ctrl + Shift + F10`  | 运行上下文配置                        | 运行当前文件或方法（如单元测试）。           |

### 6. **版本控制相关快捷键**
这些快捷键用于 Git 或其他版本控制系统（需启用 VCS 插件）。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Alt + BackQuote`     | VCS 快速操作                          | 打开版本控制操作弹出菜单（如 Commit、Push）。|
| `Ctrl + K`            | 提交更改                              | 打开提交对话框，提交代码到本地仓库。         |
| `Ctrl + Shift + K`    | 推送更改                              | 推送本地提交到远程仓库。                     |
| `Ctrl + T`            | 更新项目                              | 从远程仓库拉取最新代码（Pull）。             |
| `Ctrl + Shift + A`    | 查找操作                              | 查找并执行 IDE 中的操作或命令。              |

### 7. **窗口与视图管理**
这些快捷键用于切换视图、管理窗口布局。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Alt + 1`             | 项目视图                              | 打开或聚焦到 Project 视图。                  |
| `Alt + 2`             | 收藏夹视图                            | 打开或聚焦到 Favorites 视图。                |
| `Alt + 3`             | 查找视图                              | 打开或聚焦到 Find 视图。                     |
| `Alt + 4`             | 运行视图                              | 打开或聚焦到 Run 视图。                      |
| `Alt + 5`             | 调试视图                              | 打开或聚焦到 Debug 视图。                    |
| `Alt + F12`           | 打开终端                              | 打开 Terminal 窗口。                         |
| `Ctrl + Tab`          | 切换文件                              | 在打开的文件标签间切换。                     |
| `Ctrl + Shift + T`    | 重新打开最近关闭的文件                | 重新打开最近关闭的文件。                     |

### 8. **重构相关快捷键**
这些快捷键用于代码重构，快速修改代码结构。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Ctrl + Alt + Shift + T` | 重构菜单                           | 打开重构选项菜单（如提取方法、变量等）。     |
| `Ctrl + Alt + M`      | 提取方法                              | 将选中的代码提取为独立的方法。               |
| `Ctrl + Alt + V`      | 提取变量                              | 将表达式提取为变量。                         |
| `Ctrl + Alt + F`      | 提取字段                              | 将表达式提取为类字段。                       |
| `Ctrl + Alt + C`      | 提取常量                              | 将表达式提取为常量。                         |
| `Ctrl + Alt + P`      | 提取参数                              | 将表达式提取为方法参数。                     |
| `Shift + F6`          | 重命名                                | 重命名变量、方法或类（支持全局替换）。       |

### 9. **其他实用快捷键**
一些杂项但非常有用的快捷键。

| **快捷键**            | **功能**                              | **说明**                                      |
|-----------------------|---------------------------------------|----------------------------------------------|
| `Ctrl + Shift + A`    | 查找操作                              | 输入命令或操作名称，快速执行。               |
| `Ctrl + Alt + S`      | 打开设置                              | 打开设置窗口，调整 IDE 配置。                |
| `Ctrl + W`            | 选择代码块                            | 按层次选择代码块（重复按增加选择范围）。     |
| `Ctrl + Shift + W`    | 缩小选择范围                          | 缩小代码选择范围。                           |
| `Ctrl + Alt + T`      | 环绕代码                              | 用 `if`、`try-catch` 等结构环绕选中的代码。  |

### 10. **自定义快捷键**
Android Studio 允许自定义快捷键：
1. 打开 `File -> Settings -> Keymap`。
2. 在右侧查找操作，点击对应的快捷键进行修改。
3. 支持导入其他 IDE（如 Eclipse、IntelliJ IDEA）的快捷键方案。
