### 标签栏
在小程序全局配置文件 app.json 中添加 tabBar 配置项即可实现标签栏配置。 tabBar 配置项的属性表如下：  
|属性|类型|必填|说明|
|:-:|:-:|:-:|:-:|
|color|HexColor|是|标签栏上文字的默认颜色，仅支持十六进制颜色|
|selectedColor|HexColor|是|选中时的文字颜色，仅支持十六进制|
|backgroundColor|HexColor|是|标签栏的背景颜色，仅支持十六进制|
|borderStyle|string|否|标签栏上边框的颜色，仅支持 black 和 white |
|list|Array|是|标签栏的列表|
|position|string|否|标签栏的位置，仅支持 bottom 和 top |
|custom|boolean|否|自定义标签栏|  
* **list** 数组的每个元素都是一个代表标签按钮的对象，最少配置2个标签按钮，最多5个。  
* **标签按钮属性**  
  |属性|类型|必填|说明|
  |:-:|:-:|:-:|:-:|
  |pagePath|string|是|页面路径，页面必须在 pages 数组中预先定义|
  |text|string|是|标签按钮上的文字|
  |iconPath|string|否|未选中时标签的路径，当 position 为 top 时，不显示图标|
  |selectedIconPath|string|否|选中时标签的路径，当 position 为 top 时，不显示图标|
* 示例代码
    ```json
    "tabBar": {
    "color": "#ccc",
    "selectedColor": "#ff4c91",
    "borderStyle": "black",
    "backgroundColor": "#fff",
    "list": [{
        "pagePath": "pages/index/index",
        "iconPath": "images/invite.png",
        "selectedIconPath": "images/invite.png",
        "text": "首页"
    }, {
        "pagePath": "pages/list/list",
        "iconPath": "images/avatar.png",
        "selectedIconPath": "images/avatar.png",
        "text": "联系我们"
    }]
    }
    ```
