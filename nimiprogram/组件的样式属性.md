##### input
placeholder-style：设置占位符文本的样式，使用标准的 CSS 样式规则。
placeholder-class：设置占位符文本的样式类，可以用来定义更复杂的样式。
disable-default-padding：是否禁用默认内边距，默认值为 false。
除了上述特定于微信小程序的样式属性外，你还可以使用标准的 CSS 样式直接对 <input> 组件进行样式化，例如：

background-color：设置背景颜色。
border：设置边框样式。
color：设置文本颜色。
font-size：设置字体大小。
height 和 width：设置组件的高度和宽度。
padding 和 margin：设置内边距和外边距。
text-align：设置文本对齐方式。
line-height：设置行高。

##### button
plain：布尔值，设置按钮是否为镂空样式，默认为 false。
loading：布尔值，设置按钮是否显示为加载状态，默认为 false。
disabled：布尔值，设置按钮是否禁用，默认为 false。
form-type：设置按钮在表单中的类型，如 "submit" 或 "reset"。
open-type：用于调用微信开放接口，例如获取用户信息、分享等。
除了上述特定于微信小程序的属性外，还可以使用标准的 CSS 样式对 <button> 组件进行样式化，包括但不限于：

background-color：设置背景颜色。
border：设置边框样式。
color：设置文本颜色。
font-size：设置字体大小。
height 和 width：设置组件的高度和宽度。
padding 和 margin：设置内边距和外边距。
text-align：设置文本对齐方式。
line-height：设置行高。
border-radius：设置圆角半径。
