##### 在列表中添加元素
1. **在末尾添加**
    * ` my_list.append(value)`
1. **在指定位置中插入**
    * `my_list.insert(index, value)` 在*my_list*列表的*index*处插入*value*，原来*index*及之后的元素右移一位。
    
1. **使用 extend() 方法**
    `list.extend(iterable)`: 将一个可迭代对象（如列表）中的所有元素添加到列表的末尾。
    例如：
    ```PYTHON
    my_list = [1, 2, 3]
    my_list.extend([4, 5])  # 添加 4 和 5
    print(my_list)  # 输出: [1, 2, 3, 4, 5]
    ```

1. **使用切片赋值**
    可以使用切片在列表中的特定位置插入元素。
    例如：
    ```PYTHON
    my_list = [1, 2, 3]
    my_list[1:1] = ['a', 'b']  # 在索引 1 处插入 'a' 和 'b'
    print(my_list)  # 输出: [1, 'a', 'b', 2, 3]
    ```

##### 从列表中删除元素
1. **使用del语句删除**
    * `del my_list[index]`
1. **使用pop()删除元素**
    * `a = my_list.pop()` *pop()*方法将列表最后一个元素删除，并返回该元素。
    * `a = my_list.pop(index)` 可以弹出任何位置的元素。
1. **根据值删除元素**
    * `my_list.remove(value)` 删除列表中值为value的元素，若有多个符合条件的值，只删除第一个。

##### 组织列表
1. **排序**
    * `my_list.sort()` 升序
    * `my_list.sort(reverse=True)` 降序
    * `new_list = sorted(my_list)`
    * `new_list = sorted(my_list, reverse=True)`
1. **反转列表**
    * `my_list.reverse()`
    
##### 切片操作

`list[start:stop:step]`
- **start**: 切片开始的索引（**包括**该索引）。如果省略，则默认为 0。
- **stop**: 切片结束的索引（**不包括**该索引）。如果省略，则默认为列表的长度。
- **step**: 步长，默认为 1。如果省略，则按顺序获取元素。
