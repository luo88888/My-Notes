1. **访问字典的值**
    * `temp = my_dictionary[key]` 键不存在会引发错误。
    * `temp = my_dictionary.get(key, x)` **x**为可选参数，当键不存在时返回**x**，若没有指定**x**，则返回**None**。
1. **添加键值对**
    * `my_dictionary[key] = value`

1. **修改键值对**
    * `my_dictionary[key] = value`

1. **删除键值对**
    * `del my_dictionary[key]`

1. **遍历字典**
    * **遍历所有键值对**
        * `for key, value in my_dictionary.items():` 方法items()返回一个键值对列表。
    * **遍历所有键**
        * `for key in my_dictionary:`
        * `for key in my_dictionary.keys():`
    * **遍历所有值**
        * `for value in my_dictionary.values():`

1. **清空字典**
    * `my_dic.clear()`
    * `my_dic = {}`
