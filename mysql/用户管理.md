#### 1. 管理用户


* 查询用户信息

    mysql.user 表是 MySQL 中存储用户账户和权限信息的关键系统表。它包含了用户账户的基本信息以及全局权限，可以通过该表查询用户信息，例如：
    ```sql
    select User, Host from mysql.user;
    ```
    mysql 表存储了以下信息：

    - **（1） 用户账户信息**
        - **User**：用户名。

        - **Host**：允许用户连接的主机名（如 localhost 或 % 表示任意主机）。

        - **authentication_string**：用户的密码哈希值（MySQL 5.7 及以上版本）。

        - **password_expired**：密码是否过期。

        - **password_last_changed**：密码最后一次修改的时间。

        - **password_lifetime**：密码的有效期（天数）。

        - **account_locked**：账户是否被锁定。

    - **（2）全局信息**
        - **权限字段**：每个权限对应一个字段，值为 Y（有权限）或 N（无权限）。例如：

            - **Select_priv**：是否有 SELECT 权限。

            - **Insert_priv**：是否有 INSERT 权限。

            - **Create_priv**：是否有 CREATE 权限。

            - **Shutdown_priv**：是否有关闭服务器的权限。

            - **等等**。
    - **（3）其它信息**
        - **ssl_type**、**ssl_cipher**、**x509_issuer**、**x509_subject**：与 SSL/TLS 加密相关的信息。

        - **max_questions**、**max_updates**、**max_connections**、**max_user_connections**：资源限制。

* 创建用户
    ```sql
    CREATE USER '用户名'@'主机名' IDENTIFIED BY '密码';
    ```

* 修改用户密码
    ```sql
    ALTER USER '用户名'@'主机名' IDENTIFIED WITH mysql_native_password BY '新密码' ;
    ```

* 删除用户
    ```sql
    DROP USER '用户名'@'主机名' ;
    ```

```sql
--  创建用户zhangsan, 只能够在当前主机localhost访问, 密码123456
create user 'zhangsan'@'localhost' identified by '123456';
--  创建用户zhangsan, 可以在任意主机访问该数据库, 密码123456
create user 'zhangsan'@'%' identified by '123456';
--  修改用户zhangsan的访问密码为1234
alter user 'zhangsan'@'%' identified with mysql_native_password by '1234';
-- 删除 zhangsan@localhost 用户
drop user 'zhangsan'@'localhost';
```
