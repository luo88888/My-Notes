## MySQL 基础

### 1 启动和登录 MySQL 服务器

#### 1.1 启动 MySQL 服务器

启动 MySQL 服务器即启动 MySQL 服务。配置 MySQL 服务器时一般设置为开机启动。启动或关闭 MySQL 服务可以在 Windows 操作系统的服务窗口实现。

1. **在服务窗口启动**
    `Win + R`，输入`services.msc`回车，进入服务窗口，在这里可以看到 MySQL 服务，可以启动和关闭服务，并且可以更改启动类型。

1. **在命令行窗口启动**
    在命令行可以使用 `net` 命令启动或关闭 MySQL 服务。
    以**管理员**身份进入命令行窗口，输入命令`net start 服务名`可启动对应服务。输入 `net stop 服务名` 可关闭对应服务。

#### 1.2 登录 MySQL 服务器
1. **通过命令行登录**
    进入命令行，输入`mysql -u root -p` 回车，根据提示输入密码，出现 `mysql>`提示符，表示登录成功。

### 2 MySQL 语言

SQL 命令不区分大小写

#### 2.1 SQL 的命令动词
SQL 功能强大，可以完成数据定义、数据查询、数据操纵和数据控制功能，但其核心功能只需要用到9个动词，如下：
SQL 功能|命令动词|SQL 功能|命令动词
:-:|:-:|:-:|:-:
数据定义|CREATE、DROP、ALTER|数据查询|SELECT
数据操纵|INSERT、DELETE、UPDATE|数据控制|GRANT、REMOVE

#### 2.2 MySQL 语言的功能
MySQL 语言以标准 SQL 为主体，并进行的拓展。MySQL 数据库所支持的 SQL 功能如下：
1. 数据定义（DDL）
    DDL 用于对数据库和数据库中的各种对象，进行创建、插入、修改等操作。
    主要使用 `CREATE`、`ALTER`、`DROP` 等语句实现。
1. 数据操纵（DML）
    DML 用于操纵数据库中各种对象，进行插入、删除、修改等操作。主要通过 `INSERT`、`UPDATE`、`DELETE` 实现。
1. 数据查询（DQL）
    DQL 用于从表或视图中检索数据，`SELECT` 是使用最频繁的语句之一。
1. 数据控制（DCL）
    数据控制语言（DCL）用于安全管理，即确定哪些用户可以查看或修改数据库中的数据。实现数据控制的 `GRANT` 语句用于授予权限，`REVOKE` 语句用于收回权限。
MySQL 还支持存储过程、触发器，在其中需要使用常量、变量、运算法、表达式、内置函数等语言元素，这部分不是标准 SQL 所包含的，而是为了用户编程方便而增加的语言元素。

#### 2.3 MySQL 的数据类型

数据类型定义了数据的存储格式、取值范围和占用字节数等内容。MySQL 数据类型包括`数值类型`、`字符串类型`，`日期和时间类型`、`二进制类型`等。

##### 2.3.1 数值类型
数值类型包括`整型`、`浮点型`、`定点型`三种。

- **整型**
    MySQL 支持 SQL 标准的整型——`INTEGER`（`INT`）和`SMALLINT`。
    还支持其他的，如下：
    数据类型|无符号整数取值范围|有符号整数取值范围|字节数
    :-:|:-:|:-:|:-:
    TINYINT|0~255|-128~127|1
    SMALLINT|0~65535|-32768~32767|2
    MEDIUMIN|0~16777215|-8388608~8388607|3
    INT|0~4294967295|-2147483648~2147483647|4
    BIGINT|$0\sim 2^{64}-1$|$-2^{63}\sim 2^{63}-1$|8

- **浮点型**
    数据类型|含义|字节数
    :-:|:-:|:-:
    FLOAT(m, d)|8 位精度，m 表示总位数，d 表示小数的位数|4
    DOUBLE(m, d)|16 为精度，m 表示总位数，d 表示小数的位数|8

- **定点型**
    定点型用于存储必须被保存为确切精度的数据。在 MySQL 中，`DECIMAL(m, d)` 和 `NUMERIC(m, d)` 可被视为相同的顶点数据。其中 m 表示总位数，d 表示小数的位数。
